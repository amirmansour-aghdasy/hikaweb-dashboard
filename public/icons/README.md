## نحوه ایجاد آیکون‌ها

می‌توانید از ابزارهای آنلاین زیر استفاده کنید:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://favicon.io/

یا از favicon.ico موجود استفاده کنید و آن را به سایزهای مختلف تبدیل کنید.

**نکته**: آیکون‌ها باید با پس‌زمینه شفاف یا رنگی مناسب باشند و در تمام سایزها واضح باشند.
